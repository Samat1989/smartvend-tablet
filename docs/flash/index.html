Прошивка платы реле

SmartVend Relay (ESP32) — установка по USB прямо из браузера.

  1. Подключите плату к компьютеру по USB.
  2. Откройте эту страницу в Chrome или Edge на компьютере.
  3. Нажмите кнопку ниже и выберите COM-порт платы.
  4. Дождитесь окончания записи — плата сама перезагрузится.
Браузер не поддерживает Web Serial — откройте в Chrome или Edge на компьютере (не на телефоне). Нужен защищённый доступ (HTTPS) и десктопный Chrome/Edge.

Порт не виден? Поставьте драйвер USB-UART

Скачайте драйвер своего чипа, установите, переподключите плату и обновите страницу.

CP210x — Silicon Labs (.zip) CH340 / CH341 — WCH (.exe)

После первой установки обновления приходят по воздуху (OTA), USB больше не нужен.
