# Браузерный флешер платы реле (ESP Web Tools)

Позволяет установить прошивку `esp-relay` на ESP32 **по USB прямо из браузера** —
удалённому человеку нужен только Chrome/Edge на ПК и USB-кабель, без ESP-IDF.

## Файлы
- `index.html` — страница с кнопкой установки.
- `manifest.json` — описание образа (chipFamily ESP32, один файл на offset 0).
- `relay-merged.bin` — единый образ (bootloader + таблица разделов + otadata + app).
  Генерируется скриптом, в репозиторий коммитится.

## Как обновить образ
Из корня репозитория:
```powershell
.\make-web-flasher.ps1 -Commit -Push
```
Скрипт соберёт прошивку (`idf.py build`), склеит `relay-merged.bin`, подтянет версию
в `manifest.json`, закоммитит и запушит. GitHub Pages обновится автоматически.

## Включить хостинг (один раз)
GitHub → репозиторий → **Settings → Pages → Source: `main` / `/docs`**.
После этого флешер доступен по адресу:

**https://samat1989.github.io/smartvend-tablet/flash/**

## Как пользуется человек на месте
1. Подключает плату к ПК по USB.
2. Открывает ссылку в **Chrome/Edge на компьютере** (не телефон).
3. Жмёт «Установить прошивку», выбирает COM-порт.
4. Ждёт окончания — плата перезагрузится в `esp-relay`.

При первой установке предлагается стереть флеш (`new_install_prompt_erase`) —
это важно, если на плате была другая прошивка (другая таблица разделов/NVS).

## Замечания
- Web Serial работает только в **десктопных Chrome/Edge** и только по **HTTPS**
  (GitHub Pages подходит). Firefox/Safari и телефоны — нет.
- Если COM-порт не виден — поставить драйвер USB-UART (CP210x / CH340).
- После первой установки обновления идут по **OTA**, USB больше не нужен.
