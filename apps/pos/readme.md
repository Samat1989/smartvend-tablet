# POS-Терминал (MicroMarket)

Программное обеспечение для Android-планшета, управляющее витриной микромаркета, процессом оплаты и выдачей товара.

## Основные функции
- **Умная витрина**: Отображение товаров с актуальными остатками из облака.
- **Kaspi QR**: Интеграция с платежным шлюзом SmartVend для приема оплат.
- **Удаленное управление**: Мгновенное открытие замка по команде владельца через WebSockets.
- **Сервисный режим**: Скрытое меню для настройки и тестирования оборудования.

## Технические детали

### Стек технологий
- **Язык**: Kotlin
- **Интерфейс**: Material Design 3 (Material Components)
- **Связь**: OkHttp3 (HTTP + WebSockets)
- **Hardware**: USB-Serial для управления реле.

### Управление реле (ESP32)
Приложение общается с контроллером реле через USB-порт:
- **Скорость**: 115200 бод
- **Инициализация**: При открытии порта выставляются сигналы DTR/RTS (необходимо для инициализации ESP32). Программная задержка 1с перед отправкой команд.
- **Команда открытия**: Отправляется ASCII-строка `"1\n"`.

### Интеграция с Supabase
- **REST**: Используется для поиска UUID микромаркета и обновления статуса команд.
- **Realtime (WebSocket)**: Постоянное слушание таблицы `commands`. При появлении новой записи с `command_type: 'open'`, терминал открывает замок.
- **Polling Fallback**: Резервный опрос базы каждые 15 секунд на случай обрыва WebSocket-соединения.

## Настройка приложения
1.  При первом запуске (или через Сервисный режим) необходимо указать:
    - **Номер аппарата**: ID из базы данных (таблица `micromarkets`).
    - **Секретный ключ**: API Secret для подписи транзакций.
2.  **Секретный вход**: Чтобы попасть в сервисное меню, нужно зажать заголовок «Micro Market» на главном экране на 2 секунды.

## Жизненный цикл заказа
1.  Клиент выбирает товары -> Корзина.
2.  Переход к оплате -> Выбор Kaspi QR.
3.  Генерация QR-кода (запрос к SmartVend).
4.  Опрос статуса платежа.
5.  При успехе -> Вызов инструмента `UsbController.openDoor()`.
6.  Отправка данных о продаже в Edge Function `process_kiosk_sale`.
7.  Автоматическое обновление остатков в облаке.
